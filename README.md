# Easy Web Progress

This is a very simple project for beginners.

You can use this repository to:
- Write daily progress notes
- Practice editing files
- Learn how a small project works
- Build a habit of daily GitHub commits

How to use:
1. Open any file
2. Add a small update
3. Save the file
4. Commit your change

Goal:
Improve a little every day.

Notes:
Write your daily progress below.
